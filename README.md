# WanAndroid

[玩Android 开放API](https://www.wanandroid.com/blog/show/2)

根据 Google 新推荐的 MVVM 应用架构模式进行开发，使用了 Kotlin + Retrofit2 + Jetpack( DataBinding, LiveData, ViewModel, Navigation, Paging ) 等技术
